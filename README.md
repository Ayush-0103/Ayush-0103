### AI resume reconstruction, end to end.

- Real-time ATS compatibility scoring
- LLM-driven content rewriting — not keyword stuffing
- LangGraph pipeline: parse → score → rewrite → validate
- Live, editable output inside the app itself
- Architected as a real product, not a weekend script
